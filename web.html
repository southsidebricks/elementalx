Web Development. What is it?

Web Development is the creation of websites, which takes some skill, some creativity, and plenty of time. HTML and CSS are the first languages you should become familiar with, and are really not too difficult, especially if you're genuinely excited to make your first web page.

Keep in mind that this site was created by Year Up students just like you!

The preferred web browsers for you to use are Chrome or Firefox. For your own sake, stay away from Internet Explorer!
