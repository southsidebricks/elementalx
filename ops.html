Operating Systems

Identify and be familiarized with the most popular systems

An Operating System provides an interface between users and applications and between applications and hardware devices. You've may have heard of a few of them:

  • Windows 10
  • Mac OSX
  • iOS
  • Linux
  • Unix

The differences? Well, each has a different GUI, or Graphical User Interface, which is just the way things appear on a screen for users to navigate. Some Operating Systems are built for specific machines, but an OS like Linux is open source, and can be downloaded for free on any computer. Different OS's have different functionalities, but like a variety of car brands, they still generally sell you the same thing.

Virtual Machine

Running a new OS

Can you have two Operating Systems on a single computer?
With a Virtual Machine, you can.
A Virtual Machine is an emulator that you can have on your computer. It is basically a computer inside your computer, so you can have a Mac with a Virtual Machine running Windows (giving you access to two operating systems on a single computer).

IRA Practice

True or False practice questions

1. A virtual machine does not use the processor of its host machine

False.

2. The virtual machine is comprised of a set of specification and configuration files and is backed by the physical resources of a host machine.

True.

3. A virtual machine uses the CPU, memory, storage, and kvm of the host machine

True.

4. Only Windows Operating System allows virtual machines to be used.

False.

5. Windows can handle up to four partitions on a single hard drive

True.
