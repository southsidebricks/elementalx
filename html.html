Start Writing Code

Notepad, Sublime, Atom, Dreamweaver

The first thing you should download is a Text Editor, which is basically an application that you can write code in that can then be opened in a web browser. Notepad is pre-installed on Windows computers and is a basic and easy way to start. This website was built using Sublime Text editor.

Elements

Tags and syntax

Writing code involves a lot attention to detail. HTML is the language used to put raw content onto a website in a hierarchical order, meaning that important stuff is big, and less important stuff is smaller. This will be explained in the Structure section.

Tags are the ingredients you need to cook up a good website. They are written inside of angled brackets like this <p>.Perhaps the most basic tag to remember is the <p> tag. Most HTML tag names (in this case, p, for paragraph) are relevant to its purpose. For example, the <button> tag will create a button, the <img> tag will display an image, and the <nav> tag is used for creating a navigation bar.
Almost all tags have an opening and closing tag. A house has a front door and a back door, and it is important to have both properly closed so that nothing goes wrong.

If I want to write code to create a paragraph that says "Hello!", it would look like this:
<p>Hello!</p>
As you can see, there was an opening <p> tag and a closing tag, and the brackets where closed properly. Why is there a backslash in the closing tag? Well, all closing tags need the backslash to differentiate them from opening tags. Make sure you always use a backslash in your closing tags, or chaos will consume your website.

Structure

How to write code

So, let's figure out what your first website will look like.
You don't just start writing <p> tags all over the place, there is structure, or, organizational steps, that must be taken first.

Let's create a quick website.

  • Create a new file, and save it as index.html

What are we seeing here?

  1. The <!DOCTYPE html> tag declares the type of document and language we are using. There's a ton of languages out there, so the computer has to know which one it's reading.
  2. The <html> tag is wrapped around everything, so that web browsers will know that this is an HTML page.
  3. Everything that goes inside the <head> will not be visible to users when the website is displayed.
  4. The <meta> tag provides information for web browsers and search engines
  5. The text in the <title> is what will display on the tab when you open a new window.
  6. The <body> tag contains everything that will be visible to users on the webpage.

These tags are absolutely necessary for any webpage. All other tags are optional. Once you have this structure set up, you can begin to add content inside the <body> to be displayed on your webpage.
To see what the site would look like on a web browser, right click + ctrl and Open in Browser.

Glossary

Tags and extra info

Here are quick explanations for things you can use when building an HTML website:

  • To strikethrough text like this, you use the opening and closing <s> tags.
  • HTML code ignores if you use more than 1 space between words or letters. This means that <p>Hello_______world</p> will still display as <p>Hello world<p>.
  • Use the <i> tag to make words italic .
  • There are 6 different levels of headings, to make text bigger and more important in the structure of a web site.

    h1 is the largest,

    and
    h6 is the smallest.
  • If you want to add a subscript2 or superscript2, you use the <sub> and <sup>.
  • If you want to reference a piece of work, like a book or a film, you use the <cite> tag.
  • To create a line break in the middle
    of a sentence or
    paragraph, you use the <br /> tag.
  • You can create an ordered list with grocery items like this:
    1. eggs
    2. milk
    3. pizza
    You'll be using the <ol> opening and closing tags. However, you also need opening and closing <li> tags for each item. In code, it should look like this:
    <ol>
    <li>eggs</li>
    <li>milk</li>
    <li>pizza</li>
    </ol>
  • You use the same structure for an unordered list (<ul>), which uses bullet points instead of numerical listing, just like this glossary list.
  • If you want to add a link to another website, the code is <a href="www.nfl.com">LINK TO NFL</a>
