Combining CSS with HTML

Cascading Style Sheets

HTML puts the content on the page and CSS adds color, style, and layout. We'll walk through how to make a basic website, so you'll need to take a few steps before we write CSS.

  1. Create a new folder and save your index.html file inside of it.
    • You need the main structure set up in your index.html file. See HTML course for help.
    • Add <h1>Hello world!</h1> within the body tags so we can see it displayed in the browser.

  3. In your text editor, create a new file and save this one as style.css. Make sure you save it to the same folder containing index.html. Your folder should look something like this:

    4. Side note: if you want to add images to your website, you should create an images folder inside the root folder with the index and style sheet.

  4. The next step is to link the index.html and style.css sheets together.
    • Inside of the <head> tags in your index.html file, add a <link> tag.
    The <link> tag should look exactly like this:
    • <link rel="stylesheet" type="text/css" href="style.css">

Elements

Tags and syntax

Just like HTML, CSS has its own syntax. The way that CSS works is that you have to reference a tag from your HTML, and then you have to style it.

For example:
Let's say that you want the words "Hello world!" to be green. Here's what you do.

  1. Go to the blank CSS sheet in your text editor.
  2. In style.css, we have to reference the words we want green. They were written inside of <h1> tags, so we can start with simply typing the name of the tag. The only thing in your style.css sheet is now:
    h1
  3. CSS uses brackets just like HTML, but they are curly brackets {}. Once the tag (h1) is referenced, we can set up our styling by opening and closing the brackets. The only thing in your style.css sheet is now:
    h1{}
  4. Just like HTML, CSS ignores multiple spaces in a row, and it also ignores line breaks. For visual clarity, move the closing curly bracket down before we start styling. The only thing in your style.css sheet is now:
    h1 {

    }
  5. We want to change the color of the words inside the <h1> tags, so the syntax for this is color: green; The only thing in your style.css sheet is now:
    h1 {
    color: green;
    }
    In CSS, the first selector, being color:, determines what will be styled. There is a colon to declare the beginning of the styling. Everything after the colon is the style, and to declare the end of that statement, CSS requires a semi-colon, so it looks like color: green;
  6. Why does CSS require a semi-colon? Because more than one type of styling can be done.
    To change the font, you can use font-family and font-size. Add this extra styling to the h1.
  7. Your style.css sheet should look like:
    h1 {
    color: green;
    font-family: arial;
    font-size: 25px;
    }
    and your index.html should show up as

    Hello world!

Structure

How to organize your CSS

CSS stands for Cascading Style Sheets, and the cascading means that it goes from TOP to BOTTOM. So, here's an example to keep in mind.

Let's say that you have this:
h1 {
color: green;
font-family: arial;
font-size: 25px;
}
So, you add other code, and lower in your stylesheet, you want to make an adjustment. You add:
h1 {
color: blue;
font-size: 15px;
}

So here's what'll happen:
Since CSS is Cascading, it'll read the code for h1 that you typed first. When it goes down on the stylesheet and runs into the second h1 code, the second occurence will overwrite the first, and the font-size will change to 15, and the color will change to blue.
What won't change is the font-family, because the second occurence did not make any new adjustments, so the arial will stay in effect.

Best practices for writing CSS are that the tags that you reference in CSS should be in the same order that your HTML is written, so body styling should go near the top, and you shouldn't style the <p> or <img> until later.

Glossary

Syntax and extra info

You don't have to memorize every single style and style reference, just Google things you don't understand. Here is a list of useful tools and tips when styling an HTML document.

  • Two very useful attributes in CSS are classes and id's.
    • A Class can be added to any tag, so that you can apply CSS to multiple elements.
    • An id can be added once to any element, because just like a phone number or social security number, there can only be one. Both class and id should be non-numerical characters.
    Here is an example of what a class in a <p> tag would look like:
    index.html
    <p class="blue">This is a paragraph.</p>
    <p id="red">This is another paragraph.</p>
    • <p class="blue">This is the third paragraph.</p>
    style.css
    .blue {
    color: blue;
    }
    #red {
    color: red;
    }

    Why is it .blue and #red?
    Well, in CSS you can reference any tag by just typing it in like we did with <h1> in the Structure section. However, to tell the computer that you're working with a class instead of a tag, you put a period before the name of the class, hence, .blue, and a pound sign (#) before the name of the id.

    The end result of the 3 paragraphs should look like this in the browser:
    This is a paragraph.
    This is another paragraph.
    This is the third paragraph.
  • Although most styling should be done in the style.css sheet, CSS can be written directly on the HTML document like this:
    • The first way is by going to the <head> of your HTML file, and using <style> tags. It will look something like this:
      <style>
      p {
      color: blue;
      }
      </style>
    • The second way is inline styling, which uses classes and the <span> tag. Here's an example:
      <p>Today, I went to the <span style="color: blue;"> grocery store</span> and bought some eggs.</p>
    The <span> tag does nothing by itself, but by adding the style attribute, it becomes a styling tool in HTML.
  • Using a selector in CSS, you can style multiple elements at once by separating them with commas. This looks like this:
    h1, h2, h3 {
    font-family: arial;
    }
  • If you want to apply a style to ALL elements on a page, you use the * selector, like this:
    * {
    color: red;
    }
  • When changing text color or background color, you can use basic color names like blue, green, red, or black. However, using hexadecimals is best practice. This looks like this:
    h1 {
    background-color: #33B76C;
    }
    Check out this website that gives you exactly what you need when looking for a color.
